# Agents Overview

Este projeto utiliza uma abordagem baseada em agentes para organizar responsabilidades dentro da aplicação.

## Objetivo

Separar responsabilidades para garantir:
- Escalabilidade
- Manutenção facilitada
- Código limpo e modular

## Agentes

### Validation Agent
Responsável por validar entradas do usuário.

### UI Agent
Responsável pela interface e experiência do usuário.

### Calculation Agent
Responsável pelas regras de cálculo.

### History Agent
Responsável pelo armazenamento e recuperação de histórico.

### Currency Agent
Responsável pela seleção e formatação de moeda.

## Benefícios

- Melhor organização do código
- Facilidade para adicionar novas funcionalidades
- Redução de bugs
- Clareza na arquitetura do sistema
