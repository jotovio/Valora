# Calculation Agent

## Objetivo
Executar os cálculos matemáticos das calculadoras da aplicação.

## Responsabilidades
- Processar inputs validados
- Realizar cálculos corretos
- Retornar resultados formatados

## Calculadoras incluídas
- Juros simples
- Juros compostos
- Porcentagem
- Investimentos

## Implementação
- Uso de JavaScript puro
- Utilização de parseFloat()
- Formatação com Intl.NumberFormat

## Regras
- Não lidar com validação de inputs
- Não controlar interface
- Receber apenas dados já validados

## Evolução

### v1
Cálculos básicos

### v2
Formatação de valores

### v3
Melhoria de precisão e tratamento de edge cases
