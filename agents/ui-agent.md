# UI Agent

## Objetivo
Garantir uma interface moderna, consistente e responsiva.

## Responsabilidades
- Padronizar inputs, botões e cards
- Gerenciar tema claro/escuro
- Aplicar animações suaves
- Garantir responsividade (desktop e mobile)

## Funcionalidades
- Alternância de tema
- Feedback visual (hover, active, focus)
- Layout adaptativo
- Transições suaves entre telas

## Regras
- Não alterar lógica das calculadoras
- Manter consistência visual global
- Priorizar legibilidade e UX
