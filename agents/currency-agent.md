# Currency Agent

## Objetivo
Gerenciar a seleção e exibição de moedas na aplicação.

## Responsabilidades
- Permitir seleção de moeda
- Atualizar símbolo exibido
- Manter consistência visual

## Moedas suportadas
- Real (BRL)
- Dólar (USD)
- Euro (EUR)
- Libra (GBP)
- Iene (JPY)
- Franco Suíço (CHF)
- Dólar Canadense (CAD)
- Dólar Australiano (AUD)
- Yuan (CNY)
- Rúpia Indiana (INR)

## Implementação
- Select de moedas
- Integração com Intl.NumberFormat

## Regras
- Não realizar conversão de valores
- Apenas alterar exibição
