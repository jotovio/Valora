## 🇧🇷 Sobre o Projeto

O **Valora** é uma aplicação web composta por múltiplas calculadoras financeiras e matemáticas, desenvolvida com **HTML, CSS e JavaScript**, com suporte de inteligência artificial para acelerar o desenvolvimento, organizar o código e melhorar a produtividade.

A aplicação foi projetada com uma **arquitetura modular e escalável**, permitindo a fácil adição de novas calculadoras e funcionalidades.

---

## 🇺🇸 About the Project

**Valora** is a web application composed of multiple financial and mathematical calculators, built using **HTML, CSS and JavaScript**, with AI-assisted development to improve productivity, organization and code quality.

## 📌 Considerações de Desenvolvimento | Development Notes

### 🇧🇷

Durante o desenvolvimento do projeto, algumas decisões foram tomadas visando agilidade na construção e entrega da aplicação.

* O versionamento foi consolidado em um commit inicial contendo a maior parte das funcionalidades implementadas, após desenvolvimento iterativo local.
* Os agentes de IA não foram estruturados formalmente desde o início, sendo utilizados de forma implícita ao longo do processo.
* A construção foi feita de maneira incremental, com múltiplos prompts e ajustes contínuos.

Essas decisões permitiram maior velocidade no desenvolvimento, porém indicam oportunidades de melhoria para futuras versões, como:

* Melhor granularidade de commits
* Estruturação explícita de agentes especializados
* Planejamento prévio da arquitetura de IA

---

### 🇺🇸

During the development of the project, some decisions were made to prioritize speed and delivery.

* Version control was consolidated into a single initial commit after iterative local development.
* AI agents were not formally structured from the beginning, being used implicitly throughout the process.
* Development followed an incremental approach with multiple prompts and continuous refinements.

These decisions enabled faster development, but highlight opportunities for improvement in future iterations, such as:

* Better commit granularity
* Explicit agent structuring
* More upfront architectural planning
